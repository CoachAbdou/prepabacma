# PrepaBacMa 🇲🇦

**Plateforme éducative marocaine de préparation au Baccalauréat**

> Révise · Comprends · Réussis — Ton Bac, Notre Mission.

## 🌐 Site en ligne
[prepabacma.vercel.app](https://prepabacma.vercel.app)

## 📋 Description
PrepaBacMa est une plateforme web éducative destinée aux élèves marocains de 1ère et 2ème année Baccalauréat. Elle propose :

- 📖 Résumés de cours par chapitre
- ✏️ Exercices progressifs (Facile / Moyen / Avancé)
- 🧩 Problèmes corrigés avec méthodes
- 🗺️ Examens régionaux (16 régions du Maroc)
- 🏛️ Examens nationaux (2010 → 2025)
- 🎯 Sujets types Bac originaux
- 📝 Blog & conseils de révision
